MidiAI Installer.exe

공식 설치 · 업데이트 프로그램

MidiAI Installer는 MidiAI Studio의 설치, 빠른 업데이트, 전체 설치/복구, 런타임 점검을 한 화면에서 처리하는 Windows 전용 도구입니다.

  • Installer
  • Updater
  • Runtime Manager

Setup

설치 방법

  1. 위 다운로드 버튼으로 MidiAI Installer.exe를 받습니다.
  2. 설치 파일을 실행하고 설치 경로를 확인합니다. 기본 경로는 C:\MidiAI입니다.
  3. 처음 설치하는 경우 Full Install / Repair를 선택합니다.
  4. Core → Media → Library → Runtime → Check 순서로 진행되며, 하단 설치 로그에서 상태를 확인합니다.
  5. 최종 점검이 완료되면 Open Folder로 설치 폴더를 열 수 있습니다.

Run

실행 · 업데이트 방법

  1. 이미 설치된 경우 Installer를 다시 실행한 뒤 Quick Update로 앱 파일만 빠르게 갱신합니다.
  2. 실행 오류나 파일 누락이 있으면 Full Install / Repair 또는 Update / Repair Mode로 복구합니다.
  3. 설치 후 C:\MidiAI 폴더에서 MidiAI Studio를 실행합니다.
  4. 라이선스 인증은 앱 실행 후 Google 계정으로 로그인해 진행합니다.
  5. 문제가 계속되면 Installer의 System Check와 Show HWID 결과를 1:1 문의에 첨부해 주세요.
