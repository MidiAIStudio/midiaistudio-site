# MidiAI Studio Ultimate V3

GitHub Pages에 바로 올릴 수 있는 완성형 정적 홈페이지입니다.

## 미리보기
압축을 풀고 `ko/index.html`을 열면 됩니다.

## 적용
GitHub 저장소의 기존 파일을 삭제한 뒤, 이 ZIP 안의 파일 전체를 업로드하세요.

## 다국어
- /ko/
- /en/
- /ja/

## 교체 파일
- 실행영상: assets/demo.mp4
- 원본 샘플 오디오: assets/sample-original.mp3
- MIDI 샘플 오디오: assets/sample-midi.mp3
