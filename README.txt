MidiAI Studio 자유게시판 사진/영상 첨부 V21 패치

덮어쓰기 파일:
- assets/js/app.js
- assets/css/style.css
- board-write.html

Firebase Storage 규칙:
- firebase/storage-rules.example.txt 내용을 Firebase Console > Storage > 규칙에 붙여넣고 게시하세요.

지원 기능:
- JPG / PNG / WEBP / GIF 이미지 첨부
- MP4 / WEBM 영상 첨부
- 게시글당 최대 5개
- 파일당 최대 50MB
- 드래그 앤 드롭
- 작성 화면 첨부 미리보기/삭제
- 게시글 상세 이미지/영상 표시
- 이미지 클릭 확대 보기

적용 후 확인:
- 배포 후 Ctrl+F5 새로고침
- 모바일은 브라우저 캐시가 남을 수 있어 시크릿 모드나 캐시 삭제 후 확인
